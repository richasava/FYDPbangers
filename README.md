# FYDPbangers
the greatest FYDP in ECE history

A biometric-driven gaming system that reads a player's physiology through a wearable controller and adapts an Unreal Engine 5 game's difficulty, pacing, and feedback in real time. ALIVE closes the loop: the player's physical state drives the game, and the game drives haptic feedback back to the player.

University of Waterloo — Department of Electrical and Computer Engineering Capstone Design Project (ECE 498A/B), Group 2026.04
